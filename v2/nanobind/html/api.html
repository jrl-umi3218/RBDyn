class rbdyn.DllDirectoryManager#

Bases: AbstractContextManager

Context manager for managing DLL directories on Windows. On Windows with Python 3.8+, Python doesn’t search DLL in PATH anymore We must specify DLL search path manually with os.add_dll_directory On Linux and MacOS, we can use DYLD_LIBRARY_PATH and LD_LIBRARY_PATH environment variables, or simply the rpath search for shared libraries. ref: python/cpython#87339

https://docs.python.org/3/library/os.html#os.add_dll_directory

safe_add_dll_directory(dll_dir)#

Add a DLL directory to the search path, if path exists.

Parameters:

dll_dir (str)

Python bindings for the RBDyn library
